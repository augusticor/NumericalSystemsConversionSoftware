# Numerical Systems Conversion Software

## This software is for numerical base conversion.

## Software datasheet
Design Pattern: M.V.C.
Used I.D.E: Eclipse

**Conversions between all these bases:**

### Binary to :
  - Decimal 🕙
  - Hexadecimal 1️⃣6️⃣
  - Octal 🎱
  
### Decimal to :
  - Binary ✌🏼
  - Hexadecimal 1️⃣6️⃣
  - Octal 🎱
  
### Octal to :
  - Decimal 🕙
  - Hexadecimal 1️⃣6️⃣
  - Binary ✌🏼
  
### Hexadecimal to :
  - Decimal 🕙
  - Octal 🎱
  - Binary ✌🏼
