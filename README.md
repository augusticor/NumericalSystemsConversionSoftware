# Numerical Systems Conversion Software

## This software is for numerical base conversion, the available bases are :

**Between all these :**

### Binary to :
  - Decimal 🕙
  - Hexadecimal 1️⃣6️⃣
  - Octal 🎱
  
### Decimal to :
  - Binary ✌🏼
  - Hexadecimal 1️⃣6️⃣
  - Octal 🎱
  
### Octal to :
  - Decimal 🕙
  - Hexadecimal 1️⃣6️⃣
  - Binary ✌🏼
  
### Hexadecimal to :
  - Decimal 🕙
  - Octal 🎱
  - Binary ✌🏼
